Scheduler/timer overview
This module shows the next run of scheduled IFlows. Dates/times are in UTC format. The module takes the runtime dates from deployed artifacts only.
The left tab shows the runtimes of all IFlows with timer element for the next 24h, sorted by date. The right tab shows all IFlows with have atleast one timer including their next three runs.

IFlow comparison tool / Diff utils
Select two IFlows and click the "Create diff" button, to compare both IFlows on file level. You can select from own tenant as also from remote tenants (if configured via Dashboards DIFF_REMOTE_CPI_TENANTS parameter).
Select first IFlow for comparison

Select second IFlow for comparison
Attention: If you use the diffutils on version level, the SAP CPI dashboard will do a temporary automatically version change/rollback on your IFlow. This is necessary, since SAP CPI doesn't allow direct access to older IFlow versions. Thus the dashboard will rollback your IFlow temporary to the selected version, download it for comparison/diff and then roll it back to the current active version.
